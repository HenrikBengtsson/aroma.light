# R package: aroma.light


## Installation
Package is available on [Bioconductor](http://www.bioconductor.org/packages/devel/bioc/html/aroma.light.html).  Install in R as:

```r
source('http://bioconductor.org/biocLite.R')
biocLite('aroma.light')
```
